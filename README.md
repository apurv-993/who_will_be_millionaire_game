<H1>HOW IT WORKS?</H1>
<H3>
This is a simple command-line Quiz Game developed using Python. The program stores multiple-choice questions in a list and displays them one by one to the user. For each question, the user selects an option (1–4), and the program checks whether the answer is correct. Correct answers earn ₹1000, which is added to the player's prize money. At the end of the quiz, the user can see their total winnings based on the number of correct answers.
</H3>
